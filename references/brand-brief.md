# Brand Brief

Use this file to capture reusable brand choices for A IIGC or another AIGC platform.

## Minimum Brief

Ask for or infer:
- Exact brand name and casing
- What the acronym means
- Target users
- Product category
- Desired tone
- Competitors or references to avoid
- Required formats

## A IIGC Starter Assumptions

Use these only when the user has not provided stronger direction:
- Category: AIGC generation platform
- Tone: intelligent, modern, creative, trustworthy
- Visual emphasis: generation, intelligence, structure, and creative acceleration
- Primary use: website, product UI, social avatar, GitHub or documentation showcase
- Preferred directions: geometric lettermark, fluid gradient, futuristic wordmark

## Prompt Template

Use this structure for logo image generation:

```text
Use case: logo-brand
Asset type: logo style exploration board
Primary request: Create an exploratory logo concept for the brand text "<brand name>" in a <style direction> style.
Subject: A clean vector-friendly brand board with a distinctive mark and a wordmark reading exactly "<brand name>".
Style direction: <style details>.
Composition: light neutral background, centered logo mark, wordmark underneath, small icon and monochrome previews.
Color: <palette>.
Constraints: original concept, no copied real-company logos, no watermark, no extra text except "<brand name>", clean crisp edges, readable at small sizes.
```
