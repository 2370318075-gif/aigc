# Style Directions

Use these five directions as the default logo exploration set for AIGC and AI platform brands.

## 1. Geometric Lettermark

Use when the brand needs a reliable primary logo, app icon, favicon, or compact social avatar.

Visual language:
- Precise geometry
- Letter-based mark using A, AI, IIGC, or the brand initials
- Negative space for generation, connection, or intelligence cues
- Minimal color with one strong accent

Best for:
- Main brand mark
- Product UI
- Enterprise-facing platforms
- SVG-first delivery

Avoid:
- Overly abstract geometry with no letter recognition
- Direct copies of existing AI marks
- Tiny details that disappear at small sizes

## 2. Fluid Gradient Generation

Use when the brand needs immediate visual impact and a strong AIGC feeling.

Visual language:
- Flowing ribbons, particles, waves, or generative traces
- Cyan, violet, magenta, indigo, or luminous gradient systems
- A sense of motion from input to generated output
- Soft but controlled forms

Best for:
- Website hero visuals
- Launch graphics
- Presentation covers
- Premium AIGC product positioning

Avoid:
- Pure decoration with no recognizable mark
- Excessive glow or noisy particles
- Gradients that cannot translate to flat or monochrome variants

## 3. Intelligent Node Network

Use when the brand wants to communicate model orchestration, collaboration, ecosystem, or connected generation.

Visual language:
- Nodes arranged into a deliberate symbol, letter, orbit, or system
- Clean connecting lines
- Structured network rather than random scatter
- Enterprise-ready contrast and spacing

Best for:
- AI platform or infrastructure brands
- Agent networks
- Developer tools
- Ecosystem or community brands

Avoid:
- Blockchain cliches
- Random constellation diagrams
- Lines too thin for small sizes

## 4. Futuristic Technology Wordmark

Use when the brand name itself should carry the identity.

Visual language:
- Custom wordmark with modular cuts, aligned geometry, or controlled angular details
- One or two distinctive letter treatments
- High legibility before visual tricks
- Optional compact companion mark

Best for:
- SaaS and developer-facing AI products
- International technology brands
- Website nav bars and product headers
- Cases where the acronym is important

Avoid:
- Gamer-style sci-fi type
- Illegible custom lettering
- Too many modified letters competing for attention

## 5. Light 3D / Glassmorphism

Use when the brand needs a premium showcase version while still keeping a simpler 2D mark.

Visual language:
- Simple geometric core mark
- Frosted glass, translucent depth, soft refraction, subtle edge highlights
- Cyan and violet accents
- Flat wordmark paired with dimensional symbol

Best for:
- Website hero sections
- Launch videos
- Social previews
- Investor or product decks

Avoid:
- Making 3D the only usable logo
- Excessive reflections, complex shadows, or hard-to-reproduce materials
- Forms that cannot be flattened into SVG

## Default Priority

Start with this order unless the user specifies a different goal:

1. Geometric lettermark
2. Fluid gradient generation
3. Futuristic technology wordmark
4. Intelligent node network
5. Light 3D / glassmorphism
