# Logo AI IIGC

## Workflow

Use this skill to explore several strong logo directions before choosing one for refinement.

1. Clarify the brand only when missing:
   - Brand name and exact casing
   - Meaning of acronym or name
   - Product category and audience
   - Desired tone
   - Required outputs

2. Default to five style directions unless the user asks otherwise:
   - Geometric lettermark
   - Fluid gradient generation
   - Intelligent node network
   - Futuristic technology wordmark
   - Light 3D or glassmorphism

3. For early exploration, create one concept per direction. Keep concepts distinct enough that the user can choose a strategy, not just a color variation.

4. For each concept, provide:
   - Style name
   - Visual idea
   - Why it fits the brand
   - Strengths and risks
   - Prompt or production notes for image generation or SVG work

5. After the user selects a direction, refine only that direction unless they ask for more comparison.

6. Package final assets according to the requested medium:
   - Raster preview boards for review
   - SVG-ready construction notes for implementation
   - PNG previews for GitHub, docs, or social sharing
   - Light, dark, monochrome, and icon variants when useful

## Quality Bar

Prefer logo ideas that are simple enough to become SVG, distinct at small sizes, and memorable in monochrome.

Avoid generic AI cliches such as literal brains, random circuit boards, meaningless node clouds, robot heads, and copied marks from real AI companies.

When testing with a known brand such as OpenAI, treat it as a style placeholder only. Do not copy the official logo, typography, or brand system.
