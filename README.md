#"AIGC标志技能"

一个用于生成 AIGC 平台多风格 Logo 的 Codex Skill。

## 支持的 Logo 风格

1. 几何字母标志风
2. 流体渐变生成风
3. 智能节点网络风
4. 未来科技字标风
5. 轻 3D / 玻璃拟态风
